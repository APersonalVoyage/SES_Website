# How to Generate PDF from profile.html

## Quick Steps

The file **profile.html** is now open in your browser. This is a clean, static profile designed specifically for PDF generation.

### Generate PDF (2 Steps):

**Step 1: Open Print Dialog**
- Press **Cmd+P** (Mac) or **Ctrl+P** (Windows)

**Step 2: Save as PDF**
- Destination: Select **"Save as PDF"**
- Layout: **Portrait**
- Paper size: **A4**
- Margins: **Default** or **None**
- Background graphics: **✓ Check this box** (IMPORTANT!)
- Scale: **100%**
- Pages: **All**

**Step 3: Save**
- Click **"Save"** button
- Name it: **"Shri Engineering and Steel - Company Profile.pdf"**
- Choose location: Desktop or Documents folder

## Before Generating PDF

**Update Contact Information:**

1. Open **profile.html** in a text editor (TextEdit, VS Code, etc.)
2. Search for: `[Your`
3. Replace all placeholders with your actual information:
   - `[Your Address Line 1]` → Your actual address
   - `[Your Phone Number]` → Your phone number
   - `[Your Email Address]` → Your email
   - `[Your Website]` → Your website URL

## Troubleshooting

**Q: Images not showing in PDF?**
- Make sure "Background graphics" is checked in print dialog

**Q: Colors look washed out?**
- Ensure "Background graphics" is enabled
- Try printing from Chrome or Edge (best results)

**Q: Page breaks are wrong?**
- The layout is optimized for A4
- Make sure paper size is set to A4
- Scale should be 100%
