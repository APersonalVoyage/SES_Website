## ğŸ”§ How to Make Updates

### Updating Content
1. Open the HTML file in a text editor (VS Code recommended)
2. Find the section you want to change
3. Edit the text between the HTML tags
4. Save the file
5. Commit and push to GitHub:
   ```bash
   git add .
   git commit -m "Update content"
   git push
   ```
6. Wait 1-2 minutes for changes to go live

### Updating Colors
1. Open `styles.css`
2. Find the `:root` section at the top
3. Change the color values:
   ```css
   --primary-blue: #003d82;     /* Change this */
   --accent-orange: #ff6b35;    /* Or this */
   ```
4. Save and push to GitHub

### Adding Images
1. Add image file to `Images/` folder
2. Reference in HTML: `<img src="Images/your-image.jpg">`
3. Commit and push
