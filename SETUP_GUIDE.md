### üîß Troubleshooting

**Q: Images not showing?**
- Ensure you're opening from the correct folder
- Don't move files without moving the Images folder
- Keep folder structure intact

**Q: Styling looks broken?**
- Open in a modern browser (Chrome, Firefox, Edge, Safari)
- Make sure `styles.css` is in the same folder as `index.html`
- Clear browser cache: Cmd+Shift+Delete (Mac) or Ctrl+Shift+Delete (Windows)

**Q: How do I add more products?**
- Copy an existing product section in `index.html`
- Paste it where you want the new product
- Update the content and images
- Save and refresh browser

**Q: Can I change colors?**
- Yes! Edit `styles.css`
- Search for current colors (e.g., `#1e3a8a`)
- Replace with your desired color codes
- Use a color picker tool online
